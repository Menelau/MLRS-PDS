# Code and supplementary material for the paper: MLRS-PDS: A Meta-learning recommendation of dynamic ensemble selection pipelines.

The repository is mainly divided as follows:

- The folder **LIT** contains the code from the paper "Ensembles of Locally Independent Prediction Models". Code in this folder is an adaptation of the original one from : https://github.com/dtak/lit
- The folder **FLT** contains the code from the paper "Building forests of local trees". Code in this folder is an adaptation of the original one from : https://github.com/etamponi/eole
- The folder **Results** contains additional results (base level results) obtained by the proposed technique as well as baseline methods per dataset. It also contain the classification accuracy obtained by each possible (Pool, DS) pair (total of 49 possible configurations) per dataset.
